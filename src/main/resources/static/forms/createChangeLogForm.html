Create New Liquibase ChangeLog

Before creating a new changelog, be sure to close the existing one, in case one exists.

Pressing the button will create a new Liquibase XML ChangeLog.The "Manage ChangeLog" menu item offers various options for it.

  • You can view it by clicking "View ChangeLog"
  • You can download it by clicking "Download ChangeLog"
  • You can remove the individual ChangeSets that make it up by clicking on "Remove ChangeSet"
  • You can close the ChangeLog by clicking on "Close ChangeLog". The operation is irreversible.

You can add a ChangeSet to the ChangeLog by clicking the "Generate And Add To ChangeLog" button present within the individual pages for generating scripts.
